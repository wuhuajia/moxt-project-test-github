# Moxt Test Project

This is a test repository for SandboxAgent GitHub integration.

## Description

This project is used to test the GitHub integration flow for SandboxAgent.

## How it works

1. Claude Code runs in E2B environment
2. Clones this GitHub repository
3. Makes changes based on user requirements
4. Creates a new branch and pushes changes
5. SandboxAgent retrieves changes and merges to main
